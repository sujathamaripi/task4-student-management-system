# Student Management System

📌 Project Overview
The Student Management System is a Full Stack Web Application developed using PHP and MySQL. It allows users to register, log in, and manage student records through Create, Read, Update, and Delete (CRUD) operations. This project was developed as Task 4: Real-World Full Stack Project. for the ApexPlanet Full Stack Web Development Internship.

🚀 Features

- User Registration
- User Login Authentication
- Dashboard
- Add Student
- View Student Records
- Edit Student Details
- Delete Student Records
- Logout
- MySQL Database Integration
- Responsive User Interface

⚙️ Installation Steps

1. Install XAMPP.
2. Start Apache and MySQL.
3. Copy the project folder into:
   ```
   C:\xampp\htdocs\
   ```
4. Open phpMyAdmin.
5. Create the database:
   ```
   student_db
   ```
6. Create the required tables:
   - users
   - students
7. Open your browser and visit:
   ```
   http://localhost/student-management/
   ```
